# Python ブロック崩し (Breakout)

Pygame-ce を使用して作成された、シンプルながらも機能的なブロック崩しゲームです。

## 特徴

- **カラフルなブロック**: 5色のブロックが配置されています。
- **スコアと残機システム**: ブロックを壊すとスコアが加算され、ボールを落とすと残機が減ります。
- **日本語フォント対応**: ゲーム内のテキスト（スコア、勝敗メッセージなど）は日本語で表示されます。システム内の日本語フォントを自動的に検索して使用します。
- **リスタート機能**: ゲームオーバーまたはクリア後に `R` キーで即座にリスタートできます。

## 必要要件

- Python 3.x
- pygame-ce (Community Edition) >= 2.5.0

## インストール方法

1. リポジトリをクローンまたはダウンロードします。
2. 必要なライブラリをインストールします:

```bash
pip install -r requirements.txt
```

## 実行方法

以下のコマンドでゲームを起動します:

```bash
python src/main.py
```

## 操作方法

- **← (左矢印キー)**: パドルを左に移動
- **→ (右矢印キー)**: パドルを右に移動
- **R キー**: ゲームオーバーまたはゲームクリア時にリスタート

## ゲームのルール

- パドルを使ってボールを打ち返し、画面上のすべてのブロックを破壊してください。
- ブロックを1つ壊すごとに10点が加算されます。
- ボールを画面下に落とすと残機が1つ減ります。
- 残機が0になるとゲームオーバーです。
- すべてのブロックを破壊するとゲームクリアです。
